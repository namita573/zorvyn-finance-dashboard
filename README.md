# Z-FIN | Enterprise Financial Dashboard

A professional, high-performance finance dashboard built for the Zorvyn screening assessment. This project focuses on modern UI/UX, responsive design, and Role-Based Access Control (RBAC).

## 🌟 Key Features
- **Dashboard Summary:** High-level overview of Net Liquidity, Revenue, and Outflow.
- **Smart Insights:** Data-driven observations for quick financial decision-making.
- **Visual Analytics:** Custom-built charts for Weekly Flow Patterns and Budget Breakdown (Zero-dependency implementation).
- **RBAC (Role-Based UI):** - **Admin:** Full access to view data and "Add Entry" functionality.
  - **Viewer:** Restricted read-only view.
- **Interactive Ledger:** Real-time search and filtering of transaction records.

## 🛠️ Tech Stack
- **Framework:** React.js (Vite)
- **Styling:** Tailwind CSS
- **Icons:** Custom SVG Logic
- **State Management:** React Hooks (useState)
